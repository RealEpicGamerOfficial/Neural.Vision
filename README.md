# Neural.Vision
Neural.Vision is an AI powered object and person detection system using the COCO-SSD model for "full lighting range"

REQUIRED DEVICES TO USE: (1) Webcam and (2) Internet Connected Device (w/ at least 4GB storage)

This app requires the following web permissions: (1) Webcam, (2) Automatic Downloads

Small Side Note: This is the FREE version of Neural.Vision with less features, but uses less permissions.
Another Note: For legacy hardware, just use PRO, it is $2 for the current version forver or $5/mo to always get the newest version of PRO
